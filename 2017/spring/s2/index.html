File Types:

  • Text Files
  • Binary Files

Text files

  • Each line is a string
  • Each line is terminated by \n
  • Human-readable files
  • Editable by text editors

Binary files

  • int, double, float are directly as (0, 1) stored in the file.
  • Human unreadable
  • .exe .mp3 .mpg .jpg are examples of binary files

Steps to work with files in python

  • Specifying openning file method
  • Openning file
  • Getting the file handle
  • Reading from file
  • Writing to file

File openning methods:

  • Read mode (r)
  • Write mode (w)
  • Append mode (a)
  • Writing binary mode (wb)
  • Reading binary mode (rb)

Reading from file: #1

infile = open('lines.txt', 'r')
for line in infile:
print(line)

Reading from file: #2

buffersize = 5000
infile = open('lines.txt', 'r')
buffer = infile.read(buffersize)
while len(buffer):
print(buffer)
buffer = infile.read(buffersize)

Writing to file: #1

outfile = open('lines.txt', 'w')
outfile.write("Salam")

Writing to file: #2

outfile = open('lines.txt', 'w')
print("Salam", file=outfile)

Reading from binary file

buffersize = 5000
infile = open('lines.txt', 'rb')
buffer = infile.read(buffersize)
while len(buffer):
print(buffer)
buffer = infile.read(buffersize)
